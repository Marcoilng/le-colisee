# Le Colisée - Salle Polyvalente à Lubumbashi

Site web officiel de Le Colisée, salle polyvalente d'excellence située à Lubumbashi, République Démocratique du Congo.

## À Propos

Le Colisée est un espace événementiel premium offrant des services de location de salle pour :
- Mariages et célébrations
- Conférences et événements d'entreprise
- Galas et événements privés
- Anniversaires et réceptions

## Caractéristiques

- **Capacité** : De 100 à 1000 invités
- **Services inclus** : Éclairage professionnel, sonorisation, climatisation, stationnement
- **Exclusivité** : Suite nuptielle offerte pour les mariages
- **Services additionnels** : Traiteur, protocole, décoration sur mesure
